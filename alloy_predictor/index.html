Alloy Property Predictor

Adjust alloying element content (wt%) and get a live yield-strength estimate from a regression model trained on composition data — the browser-side sibling of a full Random-Forest + SHAP analysis done in Python.

Linear-regression estimate, embedded from coefficients trained in train_model.py (5-fold CV R² ≈ 0.96 on the synthetic training set). The full project also fits a Random Forest and explains it with SHAP for non-linear feature interactions — see the Python script for that analysis.

Dataset is synthetically generated from physically-motivated solid-solution-strengthening coefficients (see generate_dataset.py) — a stand-in for a real experimental alloy database, used here to build and demonstrate the full ML pipeline end to end.
